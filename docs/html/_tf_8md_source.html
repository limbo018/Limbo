6 # Introduction {#Parsers_TfParser_Introduction}
7 
8 Technology file (tf) is used as a configuration file in EDA tools for different technology nodes.
9 It defines various properties like the layer mapping.
10 Currently the parser reads mapping of layer name, layer id and layer abbreviation from the technology file.
