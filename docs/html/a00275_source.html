6 # Introduction {#Solvers_Introduction}
7 
8 Solvers and API for specialized problems, such as solving special linear programming problems with min-cost flow algorithms.
9 It also wraps solvers like semidefinite programming solver [Csdp](https://projects.coin-or.org/Csdp "Csdp") and convex optimization solver [Gurobi](https://www.gurobi.com "Gurobi").
